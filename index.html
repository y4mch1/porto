ASA VR - VR Simulation For Dementia Patient Care Training

ASA VR is an immersive virtual reality simulation designed to train and evaluate communication skills in healthcare settings. Players navigate through multiple scenes, each presenting unique scenarios with NPCs where they must respond appropriately using voice input.

Key Features:

• Voice-based interaction system that analyzes user responses in real-time
• Dynamic scoring system that evaluates communication quality
• Multiple scenario-based scenes with contextual introductions
• In-game assistant providing guidance and objectives
• Individual session tracking with user ID system
• Comprehensive performance scoring at the end of each session

Gameplay Flow: Users start by inputting their ID, then progress through various scenes. Each scene begins with contextual text, followed by assistant guidance. Players engage in voice conversations with NPCs, and their responses are evaluated as good or bad, affecting their overall score. The simulation continues until all scenes are completed, culminating in a final performance report.

Babi Ngepet - Indonesian Folklore Game

Babi Ngepet is a unique game based on Indonesian folklore about a mystical ritual where someone transforms into a pig to steal money. This dark practice involves a supernatural connection between the transformed thief and a lit candle guarded by an accomplice.

Dual Gameplay Mechanics:

Candle Keeper Role: Protect the candle from wind and suspicious onlookers by covering it with the E key. The candle represents the mystical link to the transformed thief - if it goes out, disaster strikes.

Babi Ngepet Role: Navigate the village as a pig using WASD controls, sneaking into houses to steal before sunrise. Press Space to begin stealing, then complete the sequence (1234) correctly. Avoid bright areas and patrolling civilians to prevent getting caught.

Key Features:

• Switch between two distinct gameplay roles with Q key
• Time-based challenge system before sunrise
• Stealth mechanics with light and patrol avoidance
• Quick-time events for stealing sequences
• Strategic role-switching when alarms trigger
• Authentic Indonesian folklore storytelling

Objective: Balance between protecting the candle and successfully completing thefts across the village while managing time and avoiding detection.

ESTRI - Indonesian Horror Escape Room

ESTRI is a horror game rooted in the Indonesian myth of "pesugihan", a dark supernatural ritual for obtaining wealth through occult practices. Born from a computer graphics class project with a socio-cultural theme, ESTRI explores the rich mystical culture of Java, immersing players in Indonesian beliefs, local wisdom, and cultural mysteries.

Gameplay:

Players must escape from a haunted house by finding hidden codes scattered throughout the environment. These codes must be entered into a numpad to unlock the exit. The game features a tense time-based challenge where failure to escape before the timer runs out results in death.

Key Features:

• Atmospheric horror based on authentic Indonesian folklore
• Hidden code puzzle mechanics requiring exploration
• High-stakes timer creating constant tension
• Dangerous item system - picking up wrong objects leads to instant death
• Numpad code entry system for escape
• Cultural storytelling about "pesugihan" rituals
• Javanese mystical atmosphere and supernatural elements

Challenge: Players must carefully explore the environment, identify the correct hidden codes while avoiding cursed objects, and input the code before time runs out - all while experiencing the unsettling atmosphere of Indonesian supernatural horror.
