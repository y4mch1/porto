Our Team

"An intelligent ingredient classification system achieving >90% accuracy using Transfer Learning on MobileNetV2. Designed to help users identify food ingredients instantly for recipe recommendations."

Project Overview

Rechef is a computer vision component for a recipe recommendation app. It solves the problem of "What can I cook with these ingredients?" by automatically identifying raw food materials from camera input.

The system is built on MobileNetV2, a lightweight architecture optimized for mobile devices, ensuring fast inference times even on smartphones. The model was trained on a custom-scraped dataset of 8 common ingredient classes.

Tech Stack & Methodology

  • Core Framework: TensorFlow & Keras.
  • Base Architecture: MobileNetV2 (Pre-trained on ImageNet).
  • Technique: Transfer Learning with Fine-Tuning (Unfreezing top 50 layers).
  • Optimization: Adam Optimizer with Learning Rate Scheduler & ReduceLROnPlateau.
  • Data Source: Custom dataset collected via Web Scraping (~4000 images).

Development Pipeline

1. Data Collection & Preprocessing

Data was collected through web scraping to ensure a diverse representation of ingredients (different lighting, angles, and states).

  • Data Augmentation: Applied aggressive augmentation (Shear, Zoom, Flip, Rotation 20°) to prevent overfitting and improve generalization.
  • Resizing: All inputs normalized to 224x224 pixels to match MobileNetV2 requirements.

2. Model Architecture Strategy

I utilized a Transfer Learning approach to leverage features learned from ImageNet. To adapt the model for specific ingredients, I added a custom classification head:
